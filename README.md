Vaultly 🔐

Vaultly is a modern, secure, and responsive password generator designed to help users create strong, unpredictable passwords with ease. 
Built using HTML, CSS, and JavaScript, it combines a clean user interface with cryptographically secure password generation to improve online security.

✨ Features
🔐 Generate strong, random passwords
📏 Adjustable password length
🔠 Include uppercase, lowercase, numbers, and symbols
🚫 Exclude similar or ambiguous characters
🛡️ Password strength indicator
📋 One-click copy to clipboard
👁️ Show/Hide generated password
🌓 Light & Dark mode
🕒 Password history with local storage
📱 Fully responsive design
⚡ Fast, lightweight, and works entirely in the browser

🛠️ Tech Stack
HTML5
CSS3
JavaScript (ES6+)
Web Crypto API (crypto.getRandomValues())

🎯 Purpose

Vaultly was built to provide a simple, fast, and secure way to generate strong passwords 
while showcasing modern frontend development practices, responsive UI design, and secure browser APIs.
